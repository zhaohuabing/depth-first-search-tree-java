# depth-first-search-tree
Java implementation of depth first search for tree data structure

test case

                  A
                 /|\
                / | \ 
               B  C  D
                 /\   \
                /  \   \
               E    F   G
                        /\
                       /  \ 
                      H    I
                     
Result: A B C E F D G H I
